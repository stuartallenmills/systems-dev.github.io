Remote Adaptable Perimeter Intrusion Detection System (RAPIDS)

Our team was responsible for the video sensors, video processing video object detection, video object classification using sophisticated Bayesnian Networks, and tracking. We also provide overall command and control and the operator GUI. The system successfully demonstrated the core capabilities of intrusion detection, tracking, and classification on intruder scenarios conducted at Bellow's Air Force Base. The RAPIDS program was a collaboration between four Hawaiian leading technology developers; Advanced Fusion Technologies (AFT), Concentris Systems, Oceanit and Pukoa Scientific.

Airborne Real-time Cueing Hyperspectral Enhanced Reconnaissance

(ARCHER) Our team was responsible for the ARCHER airborne and ground based processing systems as well as initial test and demonstration software. The airborne processor is based on a highly ruggedized PC architecture with shock and vibration mounted high speed hard disks. A high-powered DSP coprossor was deployed to provide real-time airborne hyperspectral processing cabaility. The demonstration code consisted of a C# user interface that interfaced to optimized C and DSP C code for the hyperspectral processing. ARCHER is an aerial imaging system that produces ground images far more detailed than plain sight or ordinary aerial photography can as well as providing the most sophisticated unclassified hyperspectral imaging system available, according to U.S. Government officials.[2] ARCHER can automatically scan detailed imaging for a given signature of the object being sought (such as a missing aircraft),[3] for abnormalities in the surrounding area, or for changes from previous recorded spectral signatures.[4] It has direct applications for search and rescue, counterdrug, disaster relief and impact assessment, and homeland security, and has been deployed by the Civil Air Patrol (CAP) in the United States on the Australian built Gippsland GA8 Airvan fixed-wing aircraft.[2]

Real Time Processor

The Real-Time_Porcessor was targeted for real-time exploitation of high-resolution hyper-spectral imagery in a military environment. It was comprised of 8 cards with 4 DSP processors each, for a total of 32 DSPs, that communicated over a propietary high speed "overhead" buss. The host processor ran VxWorks. Additional cards were required for IO and bulk memory. Obtaining optimal performance required careful distribution of the tasks amongst the processors taking into account bandwidth issues, and with particular need to ensure that cache thrashing was minimized. The RTP was sucessfully flown by the Navy and demonstrated real time detection performance.

Airborne Intelligence, Reconnaissance and Surveillance (AIRS)

AIRS development began at the end of 2008. The product developed a detection and classification algorithm suite for deployment on smaller UAS platforms. In the course of the AIRS program, Pukoa was given the opportunity to collect real imagery data with a RQ-11 Raven in Sattahip, Thailand. This imagery was then used to develop and demonstrate automatic target detection and classification algorithms. A follow-on program, Enhanced Airs, was awarded to deploy the demonstrated capability on a small class UAV, and to upgrade to overall cabality by providing geo-referencing and advanced tracking. Unfortunately restrictions on UAV flights in US airspace required modifications to the program to support further analysis and algorithm enhancement.

Vista Mission Playback

To enhance a two-dimensional mission playback cabability we were contracted to provide an exisitng mission playback capability an new UI based on NASA's "open-source" WorldWind java SDK, which provides a three-dimensional Google Earth type capability. An initial version was demonstrated and delivered. In addition to the Java implementation of the UI, C and FFMEG were used to support playback of arbitary video formats in sync with the mission playback via a Java-C interface.
